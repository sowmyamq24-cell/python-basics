# Python Basics

## About
Foundational Python practice covering core programming concepts — the building blocks used throughout data analysis work (pandas, automation, scripting).

## Files
| File | Topics Covered |
|------|-----------------|
| `01_variables_and_operators.py` | Variables, data types, arithmetic/comparison/logical operators, string operations |
| `02_control_flow.py` | if-elif-else, for loops, while loops, break/continue, nested loops |
| `03_functions.py` | Functions, default parameters, multiple returns, *args, recursion, lambda functions |
| `04_data_structures.py` | Lists, tuples, sets, dictionaries, list comprehensions, dictionary looping |

## Tools Used
- Python 3

## How to Run
```bash
python 01_variables_and_operators.py
python 02_control_flow.py
python 03_functions.py
python 04_data_structures.py
```
